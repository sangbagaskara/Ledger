Bab 3 · Sinkronisasi Cloud

Data Anda aman di Google Sheets.

Sambungkan Ledger ke Google Apps Script web app untuk backup otomatis ke Google Sheets pribadi. Tidak ada server pihak ketiga, tidak ada langganan.

belum tersambung
↑ push

Push lokal ke Sheets

Kirim semua data lokal (transaksi + hutang) ke Google Sheets. Cocok untuk backup berkala atau saat berpindah perangkat.

↓ pull

Pull dari Sheets

Tarik data terbaru dari Google Sheets ke browser ini. Hati-hati: ini akan menimpa data lokal.

Cara setup Apps Script
  1. Buat Google Sheet baru, beri nama Ledger.
  2. Buka menu Extensions → Apps Script.
  3. Copy isi file Code.gs dari repo, paste ke editor Apps Script.
  4. Klik Deploy → New deployment → Web app, set akses ke Anyone.
  5. Copy URL web app, paste di field di atas, klik Simpan.
  6. Klik Tes Koneksi untuk verifikasi.
